# 2026-01-30 - Finance Helper Updates

## ✅ Added Date Support

**Problem:** Script always used current date for transactions.

**Solution:** Added date extraction for "วันที่ XX" format.

**Example:**
```bash
npm run finance:add "วันที่ 17 จากบัญชี make กาแฟ 115 บาท"
# Date: 2026-01-17
```

**Changes:**
- Extract day from "วันที่ 17"
- Remove date from text before extracting amount (fixes amount detection)
- Use `setDate()` to preserve timezone context

---

## ✅ Added Wallet Names

**Problem:** Script didn't recognize "make" wallet.

**Solution:** Added custom wallet names to detection list.

**Wallets Added:**
- `make` - Make bank account
- `cashbox` - Cashbox wallet
- `krungsri`, `kbank`, `scb`, `tmb`, `ttb` - Thai banks

**Note:** Wallet "บัญชี" removed from list to avoid conflicts.
